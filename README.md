# Week 3 - Data Visualization & Dashboard

## Project Overview

This project presents an interactive dashboard built using Google Sheets based on the Superstore Sales Dataset. The dashboard summarizes key business metrics and visualizes sales performance through multiple charts.

## Dashboard Features

### KPI Cards

* Total Sales
* Total Orders
* Average Sales
* Total Categories
* Total Regions

### Charts

* Sales by Category (Column Chart)
* Sales Distribution by Region (Pie Chart)
* Sales by Customer Segment (Bar Chart)
* Monthly Sales Trend (Line Chart)

### Interactive Elements

* Region Filter
* Category Filter
* Segment Filter

## Data Story

The dashboard provides insights into overall business performance by analyzing sales across categories, regions, and customer segments. Technology products generated the highest revenue, while the Consumer segment contributed significantly to total sales. Regional analysis highlights sales distribution, and monthly trends reveal business growth patterns that can support better decision-making.

## Tools Used

* Google Sheets
* Pivot Tables
* Charts
* Slicers
* GitHub

## Outcome

Successfully designed an interactive dashboard that transforms raw sales data into meaningful business insights using visual storytelling.
